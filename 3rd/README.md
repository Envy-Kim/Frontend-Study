## Create Vuejs Project

1. 관리자 권한으로 power shell 실행
2. Get-ExecutionPolicy 명령어를 입력하면 권한 상태가 보여짐
3. RemoteSigned의 권한이 아니라면 Set-ExecutionPolicy RemoteSigned 입력
4. 변경하시겠습니까? Y
5. 2번 내용 다시 입력하면 RemoteSigned으로 변경된 것을 확인 할 수 있다.
